## Requirements

- [node](https://pnpm.io/installation) >= 16.2.0
- [zola](https://www.getzola.org/documentation/getting-started/installation/)

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
